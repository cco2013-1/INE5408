Descrição Detalhada

Classe Cliente.

Representa um cliente na simulação de supermercados.

Construtores & Destrutores

Cliente::Cliente ( int  chegada)

Construtor do Cliente.

Cria um novo Cliente no horario de chegada, definindo o tipo de cliente (buscaMenorFila ou buscaFilaComMenosProdutos), a forma de pagamento (chegue ou dinheiro) e a quantidade e o valor total dos produtos.

Parâmetros
chegadaHorário de chegada do Cliente.

Define a semente do gerador aletório baseado no número de clientes, para evitar que os clientes gerados em sequencia tenham os mesmos valores
